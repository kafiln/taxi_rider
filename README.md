# Getting started

To start the project run the server then the client.

# Server (Nestjs)

to run the frontend application

`cd server`
`npm install`
`npm run start:dev`

The application is accessible on `http://localhost:5000/price`

# Client (React)

to run the frontend application

`cd client`
`npm install`
`npm start`

The application is accessible on `http://localhost:3000`
